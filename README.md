# Codeq Care — Landing Page

Sprzedażowy landing page usługi **Codeq Care** — abonamentowy zespół wzrostu dla e-commerce.

## Struktura

```
codeq-care-landing.html   # Główny plik strony (single-file HTML/CSS/JS)
```

## Sekcje strony

1. **Hero** — Fractional Team, nagłówek + team diagram
2. **Problem** — porównanie modeli (Freelancer / Agencja / Codeq Care)
3. **The Squad** — 4 role w abonamencie (Strateg, PM, UX, Dev)
4. **Podejście** — 3 filary DNA Codeq
5. **Victory Paths** — 3 interaktywne scenariusze wzrostu
6. **Proces** — model Pudełek (Sprintów)
7. **Pilotaż** — oferta wejściowa 30 dni
8. **Pricing** — pakiety Stability (3 200 zł) i Growth (6 400 zł)
9. **FAQ** — accordion z odpowiedziami na obiekcje

## Jak uruchomić

Otwórz `codeq-care-landing.html` bezpośrednio w przeglądarce — nie wymaga serwera ani zależności.

## Dalszy rozwój

- Podmień `mailto:hello@codeq.pl` na docelowy formularz lub link do Calendly
- Dodaj logotypy klientów w sekcji Hero (trust bar)
- Zintegruj z Webflow lub CMS po zatwierdzeniu projektu w Figma
